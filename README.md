# Energy Mix Frontend

Energy Mix Frontend is the client-side user interface for the Energy Mix application. It provides an interactive dashboard to visualize energy mix data retrieved from the backend API. The project is built using React with TypeScript and is containerized using Docker.

## Deployed Application

The live application can be accessed at the following address:

[EnergyMix](https://your-deployed-app-link.com)

Take notice that it will take about 40 seconds for the render.com server to spin up.

## Backend Repository

The backend API service that powers this frontend is developed as a separate project. Its source code can be found here:

[Energy Mix API - Backend Repository](https://github.com/maciejon/energy-mix-api)

## Technology Stack

* Framework: React
* Programming Language: TypeScript
* Styling: CSS
* Containerization: Docker
